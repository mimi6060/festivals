### 1.1 Philosophie UX

| Principe | Application |
|----------|-------------|
| **"Grand-m√®re friendly"** | Interface compr√©hensible sans formation |
| **Offline-first visible** | Indicateur r√©seau toujours visible |
| **< 3 taps** | Action principale accessible en max 3 taps |
| **Feedback imm√©diat** | Chaque action = retour visuel/sonore |

### 1.2 Palette de Couleurs

```
Primary:     #6366F1 (Indigo) - Actions principales
```

| Usage | Police | Taille |
